## Ejecutar localmente

Clonar proyecto

```bash
  git clone https://github.com/kioshiokamoto/nodejs-password-generator
```

Ir a ruta de proyecto

```bash
  cd nodejs-password-generator
```

Instalar dependencias

```bash
  npm install
```

Iniciar servidor

```bash
  node app -h
```
